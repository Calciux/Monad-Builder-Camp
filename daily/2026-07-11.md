## 学到的重点

- Monad Agentic Payment 是官方主推方向，不只是「兼容 EVM」，而是额外构建了完整支付基础设施（x402 + MPP + Facilitator）
- x402 协议让 Agent 通过 HTTP 签名即支付，不需发链上交易——Facilitator 代付 Gas + 批量链上结算
- MPP SDK 的 Pull 模式（ERC-3009 签名授权 → 服务端广播）让 Agent 不需持有 MON，只需 USDC
- Monad 高性能四支柱：Optimistic Parallel Execution → Asynchronous Execution → MonadBFT+RaptorCast → MonadDb+JIT
- 四者协同：用户 Tx → MonadBFT 排序 → RaptorCast 广播 → Async 执行 → 多核并行 → MonadDb 存取 + JIT 加速
- ERC-8183 × Monad 是天然匹配：多步 Escrow 流程在 Monad 上从 60s → 2s，微任务从亏损 → 盈利
